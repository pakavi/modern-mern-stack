# Modern MERN Stack

work in progress ☑️

## Features
- CRUD operations
- Authentication
- Error handling
- Middlewares
- Permissions
- Context
- Routing

## Environment Variables

You need to add these to **.env** file

`MONGO_URL`

`MONGO_PORT`

`JWT_SECRET`

`JWT_LIFETIME`

## Run Locally

Clone the project

```bash
  https://github.com/pakavi/modern-mern-stack.git
```

Go to the project directory

```bash
  cd modern-mern-stack
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
